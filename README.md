# Automated Member Registration

This project automates the testing of the user registration page on Basketball Englandâ€™s website using Selenium and Cucumber.

## Purpose

The goal is to test different scenarios for user registration, such as:
- Successful registration
- Missing last name
- Passwords do not match
- Terms and conditions not accepted

## Technologies Used

- Java  
- Selenium WebDriver  
- Cucumber (BDD)  
- JUnit  
- Maven  

## How to Run

1. Clone the project  
2. Run the tests using your IDE or `mvn test`  
3. The browser will open and perform the automated steps
